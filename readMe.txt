## Installation Requirements:

	1. This application requires a executing environment with Java SE 8.
	2. Make sure the data files are stored in the folder: "assignment1StartingCode\res\" if 	the absolute path is not provided.

## Instruction:

	To run the program, the user should be able enter a command via command line in the
	following case insensitive format:
	java -jar Sort.jar -ffile_name -tv -sb
	• -f or -F followed by file_name (the file name and path) with no spaces
	• -t or -T followed by v (volume), h (height) or a (base area) with no spaces
	• -s or -S followed by b (bubble), s (selection), i (insertion), m (merge), q (quick) 	or 	h (heap)
	(your choice of sorting algorithm) with no spaces
	
	Examples format:
	
	• java -jar Sort.jar -fshapes1.txt -Tv -Sb
	• java -jar Sort.jar -ta -sQ -f"assignment1StartingCode\res\shapes1".txt
	• java -jar Sort.jar -tH -F"absolute path of data file" -sB
